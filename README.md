# FitFlex – Gym Class Scheduler

FitFlex is a **mobile-first gym class scheduling web application** that allows users to view fitness classes, explore instructor profiles, and book sessions online. The project simulates a real-world booking platform for **Yoga, HIIT, and Pilates classes**.

This project was developed as part of a **case study assignment** to demonstrate front-end web development concepts using **HTML, CSS, and JavaScript**.

---

## Project Overview

The FitFlex system provides a clean and interactive interface where users can:

- View the weekly fitness class schedule
- Filter classes by difficulty level
- Check available spots in each class
- View instructor profiles
- Book a class using a registration form
- Apply a discount option during booking

The application focuses on **responsive design, user interaction, and dynamic content handling** using JavaScript.

## How It Works

1. The user opens the **Schedule Page** to view available classes.
2. The user selects a class and clicks **Book Class**.
3. The selected class is stored using **localStorage**.
4. The user is redirected to the **Booking Page**.
5. The user fills out the registration form.
6. The booking is confirmed and discount is applied if selected.
